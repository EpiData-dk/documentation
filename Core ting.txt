* Relate struktur ligger IKKE på TEpiDatafile (dvs. FIsRelated, FGlobalDefs??) fjernes.
------ Tilhører relate -------
Properties related to relate-files
BackupList: TStringList
IsRelateTop:Boolean 	True if data file is top of the relate hierarchy
IsRelateFile:Boolean 	True if data file is a related file
RelateFiles: TStringList 	List of all relate files (only found in data file with IsRelateTop=true)
RelateMothers: TList 	List of data files that has relate-children
RelateInfo: PRelateInfo 	Info regarding the relate file
HasRelate:Boolean 	True if data file contains one or more relate files
